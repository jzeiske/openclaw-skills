# OpenClaw Skills

Öffentliche Sammlung von OpenClaw Skills. Jeder Skill ist ein eigenständiges Paket mit `SKILL.md` und optionalen Ressourcen.

## Enthaltene Skills

### First Client Bundle
7 Skills, die dich von null zum ersten zahlenden Kunden in 30 Tagen bringen:

| # | Skill | Beschreibung |
|---|-------|-------------|
| 1 | **industry-download** | Lädt eine komplette Industrie in 30 Sekunden ins Gehirn |
| 2 | **ride-along** | 1.200-1.800 Wörter Tagesablauf eines Branchen-Inhabers |
| 3 | **money-map** | Filtert Probleme durch 5-Framework und findet das angreifbare |
| 4 | **client-locator** | Findet exakte Facebook-Gruppen, Subreddits, Foren für deine Nische |
| 5 | **discovery-script** | 8-12 Discovery-Call-Fragen die Kunden closen |
| 6 | **blueprint-builder** | Discovery-Call-Transkript → fertiger Build-Plan |
| 7 | **closer-deck** | 6-Slide HTML Sales-Presentation aus Discovery-Call-Notizen |

## Installation

Skills werden als `.skill`-Datei verteilt und in OpenClaw importiert.
